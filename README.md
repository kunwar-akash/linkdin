 Profile Script

This is a Python script that prints out a detailed profile of Akash Kunwar, including his technical skills, work experience, and social media links. The script is structured in a class-based format and outputs the profile in a clean and organized way when run in the console.

 Features
- Displays profile information such as bio, skills, work experience, and social media links.
- Organized in a class-based structure.
- Easy to modify and extend with additional details if necessary.

 Prerequisites

- Python 3.x installed on your machine. If you don't have Python installed, you can download it from the official website: [Python Download](https://www.python.org/downloads/).

- Visual Studio Code (VS Code) or any Python-compatible IDE.

 Installation

 Step 1: Clone or Download the Repository
- Clone the repository or download the `profile_script.py` file to your local machine.

 Step 2: Install Python
- Make sure Python 3.x is installed on your machine.
  - Download from: [Python Official Website](https://www.python.org/downloads/).
  - During installation, make sure to check the box that says **"Add Python to PATH"**.

Step 3: Set Up VS Code
- Install [Visual Studio Code](https://code.visualstudio.com/).
- Install the Python extension for VS Code:
  - Open the Extensions view (`Ctrl+Shift+X` or `Cmd+Shift+X`).
  - Search for "Python" and install the extension provided by Microsoft.

 How to Run the Script

 1. Open the Script in VS Code
- Open VS Code.
- Click on `File > Open Folder` or `File > Open` and select the folder where you saved the `profile_script.py` file.

 2. Open the Integrated Terminal in VS Code
- You can open the terminal by clicking on `Terminal > New Terminal` from the top menu or using the shortcut `Ctrl+` ` (Windows/Linux) or `Cmd+` ` (Mac).

 3. Run the Script
- In the terminal, run the following command to execute the script:
  ```bash
  python profile_script.py
  ```
  or for Python 3.x:
  ```bash
  python3 profile_script.py
  ```

 4. View the Output
- After running the script, you will see your profile printed in the console with details about your bio, skills, and social media links.

 Customizing the Script
You can easily customize the profile by modifying the values in the `Profile` class. For example:
- Add new skills to the `skills` dictionary.
- Update the `bio` to reflect new information about yourself.
- Change the `contact` and `social_links` to match your personal details.
